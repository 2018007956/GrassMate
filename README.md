# GrassMate 🌱
친구들과 GitHub 활동을 함께 보는 잔디 메이트 서비스입니다.  
로그인 후 메이트를 추가하면 오늘/7일/30일 기준 코드 변경량과 순위를 한눈에 확인할 수 있습니다.

## ✨ 주요 기능
- **Follow 탭**: 메이트별 최근 활동, 변경량, 스파크라인 확인
- **My 탭**: 내 잔디 요약, 연속 기록, 이번 주 작업 패턴 확인
- **Rank 탭**: 메이트 + 나의 변경량 순위 확인
- **알림**: Top3 순위 변동, 연속 기록 끊김 알림 지원

## 🙌 이런분께 추천해요
- 친구/팀원과 GitHub 활동을 가볍게 비교하고 싶은 분
- 내 커밋 흐름과 연속 기록을 한눈에 보고 싶은 분
- 매일 코드 습관을 꾸준히 유지하고 싶은 분

## 📌 사용 방법
1. 첫 화면에서 **GitHub 로그인** 버튼 클릭
2. 안내된 인증 페이지에서 **코드 입력 후 승인**
3. `Follow` 탭에서 GitHub 아이디를 입력해 메이트 추가
4. `My`, `Rank` 탭에서 기간(오늘/7일/30일)을 바꿔 활동 비교
5. 설정에서 자동 동기화(12시간/24시간/수동), 알림, 캐시 관리

## 📊 집계 기준
- 활동량 집계는 GitHub **공개 저장소(Public Repositories)** 기준으로 집계합니다.
- Rank 탭의 순위는 변경량(`추가 + 삭제`)을 기준으로 산정합니다.

## ⚠️ 참고 사항
- GitHub API 요청 한도 초과 시, 보통 몇 초~최대 약 1시간 대기하고, 2차 제한(secondary rate limit)인 경우에는 최소 1분 이상 대기해야 합니다.
- 토큰/메이트 목록/캐시는 브라우저 로컬 저장소에 저장됩니다.
- 공용 PC에서는 사용 후 연결 해제를 권장합니다.
