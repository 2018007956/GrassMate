무결성 확인 명령어

shasum -a 256 GrassMate.dmg SHA256 파일 다운로드

설치 방법

  1. DMG를 다운로드한 뒤 파일을 실행합니다.
  2. `GrassMate.app`을 `Applications` 폴더로 드래그합니다.
  3. 처음 실행 시 우클릭 후 `열기`를 선택해 실행합니다.
  4. GitHub 로그인 후 메이트를 추가해 사용합니다.

업데이트 방식

최초 설치는 DMG를 사용하고, 이후에는 앱 내 업데이트 또는 최신 릴리즈 DMG 재설치로 유지할 수 있습니다.

설치 중 경고가 뜨면 앱 서명/노타라이즈 상태를 릴리즈 노트에서 먼저 확인하세요.
